Este proyecto fue desarrollado como resolución de un examen técnico, cuyo objetivo es implementar un sistema simplificado de home banking que permita visualizar el saldo de la cuenta principal de un cliente y los 
últimos movimientos de su tarjeta de crédito principal. La solución contempla una arquitectura completa, incluyendo backend en .NET 8, frontend en React y base de datos en SQL Server Express 2019. Dentro del 
repositorio se incluye la documentación técnica y funcional del desarrollo, así como una carpeta de scripts SQL que contiene los archivos necesarios para la creación de la base de datos, tablas y stored procedures 
requeridos para el correcto funcionamiento del sistema.
